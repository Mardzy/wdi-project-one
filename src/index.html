Game Rules

Poker Dice, a poker style game with 5 dice, 3 rounds, versus the computer.
Up to 3 dice can be exchanged each round. Select dice you would like to keep. Highest value is 6, lowest is 1.
Click Roll dice to start the next round after you have made your selections, or hold if you want to stay with what you have.
